# プロジェクト名
FPS_Chess

## 概要
全世界で人気のあるボードゲームのチェスをモチーフにしたゲームです.
プレイヤーが操作するのは、チェスの駒ですが自駒の移動範囲内でしかプレイヤーは盤面を見ることが出来ません.

## 使用技術
- Unity.C#
- Unity
- Blender
